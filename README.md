📊 From Raw Data to Real Insights — My SQL Project 🚀
I recently completed a hands-on project on Online Bookstore Data Analysis using SQL, and it was an amazing learning experience!

💡 What I explored:

Book data (genres, pricing, stock)

Customer behavior & order patterns

Sales performance & revenue insights

Inventory management using real queries

🔍 Some interesting things I built:

✔ Identified most popular genres & top-selling books

✔ Found loyal customers with multiple orders

✔ Calculated total revenue generated

✔ Analyzed author-wise and genre-wise sales

✔ Built an inventory tracking query to monitor stock vs orders

📦 One of my favorite queries helped calculate remaining stock after fulfilling all orders, which is something used in real-world systems to avoid stockouts.

🛠 Skills I used:
SQL | JOIN | GROUP BY | HAVING | Aggregate Functions | Data Analysis







📈 This project gave me a deeper understanding of how data can drive decisions in an e-commerce environment.

I’m just getting started and excited to build more projects in Data Analytics !🚀
